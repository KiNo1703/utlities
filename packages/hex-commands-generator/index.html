Как работает кодировка:

Формат команды: 57[маска][год+день1][год+месяц1][год+день2][год+месяц2]

  • Год кодируется 7 битами (последние 2 цифры года)
  • Для дней берутся 3 старших бита года + 5 бит дня
  • Для месяцев берутся 4 старших бита года + 4 бита месяца

Информация о генераторе

Этот инструмент позволяет генерировать HEX-команды для планировщика.

Для команды чтения DLMS регистров (0066):

  • Task ID - номер задачи (1-255)
  • Период - интервал выполнения задачи в секундах
  • Задержка - задержка перед первым выполнением в секундах
  • ID журнала - идентификатор журнала в HEX (например, 1E)
  • Объекты - выбираемые объекты для чтения (2-15)
